目的

HTMLのボタンとJavascriptの動きを確かめる

内容

実行ボタンを押してください

使用コード

HTML表示 
  
  <button id="btn" onclick="changeText()">実行</button>
  <div id="text">実行ボタンを押してください</div>
JavaScript表示 
  
  function changeText() {
  const resultDiv = document.getElementById("text");
  resultDiv.textContent = "実行ボタンが押されました";
  resultDiv.style.color = "red";
  }
